# cafe-a5 (AAAAA Group)

Website company profile cafe bernuansa Minang (family friendly) menggunakan **PHP** + **Bootstrap 5**.

## Fitur
- Halaman: Home, About, Menu, Gallery, Order, Contact
- Tampilan responsif (desktop & mobile)
- Struktur PHP include (`includes/`, `pages/`, `config/`)
- Asset gambar & CSS/JS di folder `assets/`

## Cara Menjalankan (Local)

### A) Pakai XAMPP (Windows)
1. Install XAMPP
2. Pindahkan folder `cafe-a5` ke:
   - `C:\xampp\htdocs\cafe-a5`
3. Buka XAMPP Control Panel â†’ klik **Start** di **Apache**
4. Buka browser dan akses:
   - `http://localhost/cafe-a5/`

### B) Pakai Laragon (Windows)
1. Install Laragon
2. Pindahkan folder `cafe-a5` ke:
   - `C:\laragon\www\cafe-a5`
3. Klik **Start All**
4. Buka browser:
   - `http://localhost/cafe-a5/`

## Catatan Penting
- Karena ini **PHP**, project **tidak bisa jalan langsung di GitHub Pages** (Pages hanya static HTML/CSS/JS).
- Repo GitHub bisa dibuat **public** untuk pamer kode / kolaborasi, dan project tetap dijalankan via server lokal (XAMPP/Laragon) atau hosting PHP.
